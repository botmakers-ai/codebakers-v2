# RCA - Deep Root Cause Analysis

Deep root cause analysis on pasted error or current issue.

Traces data flow through codebase, finds systemic issues, fixes pattern comprehensively.

Load and execute agents/meta/error-investigator.md in Deep RCA mode (skip quick fix triage).

Investigation includes:
- Trace data flow backwards
- Check DEPENDENCY-MAP.md for state issues
- Search for similar patterns across codebase
- Find root cause, not just symptom
- Fix: immediate + upstream + pattern + prevention
- Log to ERROR-LOG.md with full analysis
