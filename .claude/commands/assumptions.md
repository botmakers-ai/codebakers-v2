# Assumptions - Show All Automatic Decisions

Show all automatic decisions made by CodeBakers with reasoning.

Read .codebakers/ASSUMPTIONS.md and display all automatic decisions documented.

Format:
```
[Date/Time] - [Decision Made]
Context: [What triggered this decision]
Reasoning: [Why this approach was chosen]
Alternatives Considered: [What else was possible]
Reversibility: [Easy / Medium / Hard to change later]
```

If ASSUMPTIONS.md doesn't exist: "🍞 CodeBakers: No assumptions documented yet. This file populates as automatic decisions are made during the build."

Purpose: Transparency into every automatic decision so you understand what CodeBakers chose and why.
